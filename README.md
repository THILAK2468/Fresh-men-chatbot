## 🌟 About The Project

IntraIQ (Freshmen Chatbot) is a powerful, local-first intelligent assistant designed to help new employees and team members navigate company policies, documents, and procedures seamlessly. Powered by **Google's Gemini AI**, this application enables intuitive Q&A over document contexts.

### ✨ Key Features

* 🤖 **Smart Assistant:** Engage with an AI that understands company context and provides accurate answers.
* 📚 **Document Context:** Queries are answered based on the provided company documents, linking direct sources.
* 🔒 **Local First:** Operates securely using your browser's local storage—no external databases required!
* 📊 **History Tracking:** Seamlessly tracks all your past queries, uploads, and connections.
* 🎨 **Beautiful UI:** Built with React, Tailwind CSS, and Lucide icons for a premium, responsive experience.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Node.js (v16 or higher)
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Get a free API Key at [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Clone the repo
   ```sh
   git clone https://github.com/THILAK2468/Fresh-men-chatbot.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Create a `.env` file in the root directory and enter your API key
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```
5. Run the development server
   ```sh
   npm run dev
   ```
